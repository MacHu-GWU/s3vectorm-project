Install
------------------------------------------------------------------------------

``s3vectorm`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install s3vectorm

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade s3vectorm
